# Learning Notes

Organize and structure your study materials effectively.

## Note Features

- Rich text editing
- Code syntax highlighting
- Image support
- Tag system

## Organization

### Folders

Create hierarchical structures:

```
Study/
  └── Computer Science/
      ├── Algorithms/
      └── Data Structures/
```

### Tags

Use tags for cross-referencing:

- #important
- #review-needed
- #exam-prep

## Integration with Flashcards

- Convert notes to flashcards
- Link related materials
- Track connected concepts
