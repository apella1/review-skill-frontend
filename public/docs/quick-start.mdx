# Quick Start Guide

Get up and running with ReviewSkill in minutes.

## Installation

```bash
git clone https://github.com/your-username/reviewskill
cd reviewskill
yarn install
yarn dev
```

## Creating Your First Flashcard

1. Navigate to the Flashcards section
2. Click "New Deck"
3. Name your deck
4. Add cards with front and back content
5. Start studying!

## Setting Up Your Study Schedule

1. Go to Settings
2. Configure your daily study goal
3. Set preferred study times
4. Enable notifications (optional)
