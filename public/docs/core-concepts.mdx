# Core Concepts

Understanding the fundamental principles behind ReviewSkill.

## Spaced Repetition

Spaced repetition is a learning technique that incorporates increasing intervals of time between subsequent review of previously learned material.

### How it Works

1. Study new information
2. Review at increasing intervals
3. System adapts to your performance

## Active Recall

Active recall is the practice of actively stimulating memory during the learning process.

### Implementation

- Flashcards force active recall
- Question-first approach
- Self-evaluation

## Metadata Tracking

We track various metrics to optimize your learning:

- Review intervals
- Success rates
- Learning patterns
